4. Difference between flex row and column

Flexbox is a layout module in CSS that allows you to create flexible and responsive layouts with ease. The main difference between flex row and column is the direction in which the elements inside the flex container are arranged.

Flex row

  1. When the flex container is set to display as a row, the elements inside it are arranged horizontally.
  2. This means that the main axis of the container is horizontal, and the cross axis is vertical.
  3. In other words, the elements are positioned side-by-side.

Flex column

  1. When the flex container is set to display as a column, the elements inside it are arranged vertically.
  2. This means that the main axis of the container is vertical, and the cross axis is horizontal.
  3. In other words, the elements are positioned on top of each other.
