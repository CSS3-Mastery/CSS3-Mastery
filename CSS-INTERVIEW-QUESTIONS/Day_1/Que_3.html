3. What is the difference between padding and margin?

Padding

  1. Padding is the space between an element's content and its border.
  2. It is used to create space inside an element.
  3. All sides at once using the shorthand property "padding".

Margin

  1. Margin is the space between an element's border and the adjacent elements.
  2. It is used to create space outside an element.
  3. All sides at once using the shorthand property "margin".
