6. Difference between absolute and relative positioning.

Absolute

  1. Position is relative to the nearest positioned ancestor element (with a position property of relative, absolute, or fixed).
  2. The element is removed from the normal document flow, and other elements no longer flow around it.
  3. The element's position is determined by the values of the top, right, bottom, and left properties, which are relative to the closest positioned ancestor.
  4. If there is no positioned ancestor, the element will be positioned relative to the body element.
  5. The element's size and position may need to be explicitly defined.

Relative

  1. Position is relative to the element's normal position in the document flow.
  2. The element still takes up space in the normal document flow.
  3. Other elements will flow around the relatively positioned element as if it were still in its original position.
  4. Position is defined using the top, right, bottom, and left properties.
  5. Element's position can be adjusted by changing the values of the top, right, bottom, and left properties.
