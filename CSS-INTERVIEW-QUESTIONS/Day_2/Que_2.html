2. What is animation delay?

Animation delay is a CSS property that allows you to specify a time period before an animation begins. It's often used to create staggered or sequenced animations, where different elements on a page start animating at different times.

Here's an example of how you can use the animation-delay property in your CSS:

For example

HTML

    <div> </div>

CSS

    div{
        width: 100px;
        height: 100px;
        background-color: red;
        animation: div 5s infinite;
        animation-delay: 3s;
    }
    @keyframes div {
        from{
            margin-left: 0;
        }
        to{
            margin-left: 200px;
        }
    }
